Why
  • 기여도: 기획 및 여러 제작 부서와의 커뮤니케이션 필요 — 70% 기여

  • 목적: 시청자의 흥미 유발 + 조회수 향상

Solution
  • 극장판 홍보 흐름을 유지하면서도 개별 콘텐츠 자체의 재미와 완결성을 갖춘 오리지널 쇼츠 기획
  • 캐릭터 성격 기반 상황극 포맷을 활용해 콘텐츠 소비 → 채널 재방문 → 구독으로 이어지는 유입 구조 설계
Performance
  • 한 달 내 조회수 1,000만 이상 콘텐츠 2편 달성 — 오리지널 쇼츠 포맷의 확산 가능성 검증
  • 시청자 참여율 69% → 72.8% 상승, 이탈률 감소
  • 댓글 내 영화 관련 언급 증가 → 자연스러운 홍보 효과 확인
Roles & Responsibilities
  • 타겟(1020세대) 및 트렌드 분석 기반 밈/패러디 콘텐츠 기획
  • 콘텐츠별 조회수·공유·팔로워 전환율 등 핵심 지표 분석 및 성과 관리
  • 데이터 기반 인사이트를 다음 콘텐츠 기획에 반영 — 트렌드 변화 빠르게 적용
Lessons Learned
  • 쇼츠는 단순 영상 편집이 아닌 타겟의 공감과 반응을 유도하는 구조 설계라는 점을 체득

Why
  • 기여도: 런칭·메인 예고편 영상 편집 및 쇼츠 콘텐츠 제작 수행

  • 콘텐츠 노출은 유지되었지만 CTR, 조회수, 구독자 증가로 이어지지 않는 구조적 한계 존재
Solution
  • 썸네일을 단순 장면 캡처에서 시청자의 호기심을 유도하는 감정·반전 중심 이미지로 전환
  • 시즌 이슈·트렌드 콘텐츠를 반영한 썸네일을 기획하여 시의성과 클릭 유도 요소를 동시에 강화
Performance
  • 입사 전/후 4개월 비교 기준 CTR 5.7% → 6.0% 상승
  • 예상 수익 $8,500 이상 증가 — 썸네일 개선이 수익과 직접적으로 연결됨을 확인
  • 구독자 증가율 약 12% 향상 — 데이터 기반 전략의 유효성 검증
Roles & Responsibilities
  • 이벤트·트렌드 이슈 기반 썸네일 기획으로 캐릭터 중심 구조 → 클릭 유도형 썸네일로 전환
  • YouTube Studio A/B 테스트를 활용해 썸네일별 CTR 및 시청 시간 성과 비교·검증
  • 콘텐츠 업로드 일정 수립 및 운영 총괄 — 썸네일 개선 전략을 전체 콘텐츠에 적용
Lessons Learned
  • 썸네일은 단순한 시각 요소가 아니라 사용자의 클릭을 유도하는 '상황 설계'라는 점을 학습
  • 개인적인 흥미보다 대중이 공감할 수 있는 보편적 상황과 감정을 반영할 때 CTR과 반응이 더 높아진다는 인사이트 확인

체리툰
체리툰 오리지널 숏폼 콘텐츠 기획·제작
2025.09 – 2026.03
트렌드 기반 숏폼 콘텐츠 전략 전환을 통해 조회수 300만+ 성과를 만든 오리지널 쇼츠 기획·제작
Why
  • 기여도: 기획 및 여러 제작 부서와의 커뮤니케이션 필요 — 70% 기여
  • 타겟: 1020세대
  • 목적: 타겟의 관심을 유도하고 조회수를 확보할 수 있는 숏폼 콘텐츠 기획·제작
Problem
  • 시즌 3 제작 지연으로 신규 콘텐츠 공급이 어려운 상황 — 기존 재편집 콘텐츠만으로는 사용자 이탈을 막기 어려운 구조
  • 반면 캐릭터 IP의 인지도는 유지되고 있어 지속적인 노출과 팬 유입을 위한 대체 콘텐츠 전략이 필요
Solution
  • 숏폼 중심 콘텐츠 운영 전략으로 전환하고 트렌드·밈·시의성을 반영한 오리지널 콘텐츠 기획
  • 캐릭터 성격 기반 상황극 포맷을 설계하여 반복 제작 가능한 콘텐츠 구조 구축
  • 콘텐츠 소비 → 채널 재방문 → 구독으로 이어지는 유입 구조를 고려해 설계
Performance
  • 4개월간 조회수 300만+ 숏폼 콘텐츠 6건 제작
  • 콘텐츠 기획 및 편집 방식 개선을 통해 시청자 평균 참여율 76.7% → 80.3% 상승
Roles & Responsibilities
  • 트렌드 및 타겟 분석 기반 콘텐츠 기획 방향 수립
  • 조회수·참여율 등 핵심 지표 분석 및 개선 방향 도출
  • 데이터 기반으로 콘텐츠 전략을 반복 개선하며 제작 구조 구축
Lessons Learned
  • 숏폼 콘텐츠는 단순 편집이 아니라 타겟과 상황을 기반으로 설계되어야 한다는 점을 학습
  • 포맷 구조에 따라 조회수와 유입 성과가 달라지며 콘텐츠 기획 단계에서 구조 설계가 핵심이라는 인사이트 도출

CJ ENM
투니버스 IP 기반 콘텐츠 기획·운영 (썸네일·SNS)
2021.05 – 2021.11
CJ ENM 투니버스 IP를 활용해 클릭을 만드는 썸네일과 콘텐츠처럼 소비되는 SNS 콘텐츠를 함께 설계. 감정·궁금증 중심 썸네일과 트렌드 기반 캐릭터 콘텐츠를 결합해 유입부터 반응까지 이어지는 콘텐츠 구조를 구축
Why
  • 기여도: 썸네일 및 SNS 콘텐츠 기획·제작 전 과정 직접 수행
  • 타겟: 1020세대
  • 목적: 클릭 → 소비 → 반응으로 이어지는 구조 설계 및 성과 개선
Problem
  • 기존 콘텐츠는 썸네일 클릭 이후 소비와 반응으로 이어지지 않는 구조적 한계 존재
  • 홍보성 콘텐츠는 광고 느낌이 강해 자연스럽게 소비되지 못하고 이탈로 이어지는 문제 발생
Solution
  • 썸네일은 감정·궁금증 중심으로 클릭 유도, SNS 콘텐츠는 트렌드 기반 상황극으로 자연스러운 소비 유도
  • 유입과 소비 각 단계에서 사용자 반응을 유도하는 구조로 개선
Roles & Responsibilities
  • 감정·궁금증 중심 썸네일 기획으로 클릭 유도 개선
  • 캐릭터 상황극 + 트렌드 반영 SNS 콘텐츠 제작으로 자연스러운 소비 유도
  • 역할별 전략 분리를 통해 클릭과 반응을 각각 최적화
Lessons Learned
  • 개인 취향보다 대중이 공감할 수 있는 상황·감정을 시각화하는 것이 중요함을 체감
  • 콘텐츠는 적절한 타이밍과 포인트 설계, 반복 노출을 통해 성과로 이어진다는 인사이트 도출
